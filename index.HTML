HTMLの要素を説明する!!!!!


  1. storong要素
    1. 説明

      2. strong要素は強い重要性のある内容、 警告や緊急性のある物事のを表示するためのものです。 ページ全体にとって非常に重要な文である可能性もあるので。 同じ表示をするb要素と同じように使ってはいけません。

    2. マークアップ例

      3.             
              <p><strong>警告:</strong>セーブデータが保存されていません、本当にによろしいですか？</p>
    3. 動作結果

      4. 警告：セーブデータが保存されていません、本当によろしいですか？


  2. abbr要素
    1. 説明

      2. abbreviationの略語で略称を示すタグです。 英語の頭字語でできた略語の場合に使用する要素です。 title属性を指定することで省力する前の言葉を示すことが可能です。

    2. マークアップ例

      3.             
              <abbr title="Hyper Text Transfer Protocol">HTTP</abbr>
    3. 動作結果

      4. HTTP


  3. kbd要素
    1. 説明

      2. 「kbd」は「keyboard」の略称であり、 ユーザーがキーボードなどから入力するテキストを示すために 使用されます。

    2. マークアップ例
      3.             
              <p>ここに<kbd>生年月日</kbd>を入力してください</p>
    3. 動作結果

      4. ここに生年月日を入力してください
